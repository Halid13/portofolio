# Dossier CV

Placez votre fichier CV au format PDF ici avec le nom :
**`Mohamed_ONIFADE_CV.pdf`**

Ce fichier sera affiché dans la section "À propos" de votre portfolio et pourra être téléchargé par les visiteurs.

## Instructions
1. Générez ou exportez votre CV en PDF
2. Renommez-le : `Mohamed_ONIFADE_CV.pdf`
3. Placez-le dans ce dossier (`cv/`)
4. La section "À propos" l'affichera automatiquement

## Mise à jour
N'oubliez pas de mettre à jour la date dans `index.html` (ligne avec "Dernière mise à jour") à chaque modification du CV.
